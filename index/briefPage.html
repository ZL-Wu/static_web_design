Content:

This is our web structure:

The goal of our web is to introduce "Tampa" city to the tourists, and help them know this city quickly. Then we divided it into three dimensions:

  • Tampa Travel: introduces attractions in Tampa. So far, we completed Busch Garden Theme Park.
  • Tampa Eat: introduces some good restaurants, such as Columbia Restaurant, Noble Rice, Sa Ri One Grill.
  • Tampa Sport: introduces the professional ice hockey team in Tampa, Tampa Bay Lightning.
    •

    In every subpage, we tried to explain theme by many topics, such as photos, introduction, videos and so on

    Usability:

    In order to let Users feel free and easy to browse. We create a uniform and concise navigation bar on interface in every page. Users can easily enter the three subpages from homepage, and easily jump to each topic in the subpage.

    We put an eye-catching photo board on every page. We believe pictures are more convincing than words. Therefore Users can quickly get an intuitive feeling on each subject

    Responsiveness:

    We want to make pages layout display same on each device. So we didn’t use flex layout to be compatible with any device but controlled the page layout in a uniform style by CSS code: .body{ width:1000px; margin: 0 auto;}
    Therefore, every page has only 1000px width and display the same at any device.

    Design:

    We designed this website based on these guidelines:

  • Simple and clean interface: try to minimize the mix of different color types.
  • Clear and easy page: clear navigation and neat block arrangment.
  • Pictures and videos: they are more intuitive than words.
    •

    Shortage

    This is not a finished product, there are still some shortages:

  • In order to easily manage framework layout, we sacrifaced compatibility layouts on different devices.
  • In the three dimensions, we still have many attractions, foods and sports that are not introduced yet.
  • Due to lack of JS skills, we have many excellent animation effects ideas that cannot be achieved.
